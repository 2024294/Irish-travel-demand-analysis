# Travel by Irish Residents: A Comprehensive Data Analysis

This project presents a comprehensive analysis of travel behavior among Irish
residents between **2000 and 2023**, focusing on **domestic vs outbound travel**
and **reasons for journey**.

The study combines **data preparation, statistical analysis, and machine learning**
to uncover travel patterns, spending behavior, and predictive insights relevant
to the tourism industry.
